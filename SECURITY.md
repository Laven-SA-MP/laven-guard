# Security Policy

LAVEN GUARD için exploit, abuse veya güvenlik açığı raporlarını sorumlu açıklama ilkesiyle paylaşın.

## Raporlama

- İletişim: security@laven.dev
- Konu başlığı: `[LAVEN GUARD SECURITY] <kısa özet>`
- Beklenen içerik: etki seviyesi, yeniden üretim adımları, PoC (varsa), önerilen düzeltme.

## Sorumlu Açıklama

- Açık detaylarını public alanda paylaşmadan önce bakım ekibine makul bir düzeltme süresi tanıyın.
- Kullanıcıları riske atacak 0-day detaylarını issue veya sosyal medya üzerinden yayımlamayın.
- Onaylandığında güvenlik düzeltmeleri sürüm notlarında kredi verilerek duyurulur.
